# 🌦️ Weather App

A modern Weather Application built with React, TypeScript, Tailwind CSS, and Zustand.

## 🚀 Features

- 🔍 Search weather by city name
- 📍 Get weather using current location (Geolocation API)
- 🌡️ Current weather details (temperature, humidity, wind, etc.)
- 📅 5-day weather forecast
- ⚡ Loading & error handling states
- 💾 Save last searched city (localStorage)
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand
- Axios
- React Router DOM
- OpenWeather API
